Detailed Description

Implements a logistic flow with population limit.

The LogisticFlow calculates the value transferred through a logistic function that simulates growth limited by environmental capacity.

The formula used is: flow = 0.3 * P * (1 - P/pmax) where P is the current value of the sink system and pmax is the maximum population.
