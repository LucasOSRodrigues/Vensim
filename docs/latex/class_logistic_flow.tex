\doxysubsection{Detailed Description}
Implements a logistic flow with population limit. 

The \doxylink{class_logistic_flow}{Logistic\+Flow} calculates the value transferred through a logistic function that simulates growth limited by environmental capacity.

The formula used is:\+ flow = 0.\+3 \texorpdfstring{$\ast$}{*} P \texorpdfstring{$\ast$}{*} (1 -\/ P/\+pmax) where P is the current value of the sink system and pmax is the maximum population.
