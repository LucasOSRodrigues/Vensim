\doxysubsection{Detailed Description}
Class responsible for executing the simulation. 

The model contains and manages a set of systems and flows. It is responsible for storing them and executing the temporal evolution of the simulation, applying the transfer rules between systems in each iteration.

void Model::\+run (\begin{DoxyParamCaption}\item[{int}]{begin}{, }\item[{int}]{end}{}\end{DoxyParamCaption})}



Executes the simulation in the specified time interval. 

Simulates the evolution of the system from the initial time to the final time, applying flows in each iteration and updating system values.


\begin{DoxyParams}{Parameters}
{\em begin} & Initial simulation time (inclusive). \\
\hline
{\em end} & Final simulation time (exclusive).\\
\hline
\end{DoxyParams}
For each time unit in the interval \+[begin, end):\+
\begin{DoxyEnumerate}
\item Calculates the value transferred by each flow
\item Reduces the value of the source system
\item Increases the value of the sink system
\end{DoxyEnumerate}

For each time unit in the interval \+[begin, end):\+
\begin{DoxyEnumerate}
\item Executes each flow and stores its return value
\item For each flow:\+
\begin{DoxyItemize}
\item Deducts the value from the source system (if not null)
\item Adds the value to the sink system (if not null) 
\end{DoxyItemize}
\end{DoxyEnumerate}
