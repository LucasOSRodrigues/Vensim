# MyVensim

Projeto da disciplina Engenharia de Software I.

## Compilar

```bash
make
```

## Executar produto

```bash
make run
```

## Executar testes funcionais

```bash
make run_funcional_tests
```

## Executar testes unitários

```bash
make run_unit_tests
```

## Executar todos os testes

```bash
make tests
```
